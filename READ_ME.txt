READ ME : 

Nous vous fournissons trois fichiers python pour ce TP de Géostatistiques: 
	- lib_gs.py: fait office de librairie de toutes les méthodes d'interpolation ainsi que leur affichage graphiques. Chaque méthode d'interpolation est définit sous la forme d'une fonction avec différents paramètres
	- tp_krigeage.py: fichier avec l'ensemble des fonctions nécessaires pour l'interpolation par krigage (notamment toutes les fonctions pour calculer les variogrammes). Nous avons préféré faire un fichier à part pour cette méthode pour rendre le code plus lisible
	- final.py: fichiers pour tester nos méthodes d'interpolation avec nos validations croisées et l'évaluation du levé avec les données de contrôle. Ce fichier importe les deux précédents en faisant appelle à leur fonction. Une seule fonction est redéfinie: celle pour effectué les comparaisons entre les données test et les données interpolées. Toutes les méthodes d'interpolation ont été testées dans le main.
